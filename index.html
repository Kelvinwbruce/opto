How to set up:
  1. On your phone: Opto → Settings → Connect backup file → save to your Google Drive folder as opto-phone-backup.json
  2. Wait a moment for Google Drive to sync to this PC
  3. Here on desktop: click Connect phone sync file and pick that same file
  4. Done. New captures auto-sync every 5 min while Opto is open, and immediately on next load.
